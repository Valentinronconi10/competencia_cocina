# Competencia de Cocina

Este proyecto simula una competencia de cocina donde 5 participantes son evaluados por 3 jueces en distintas rondas. Se calculan los puntajes, se determina un ganador por ronda y se genera una tabla de posiciones.

## Estructura del proyecto

competencia_cocina/
├── src/
│   └── cocina.py
├── notebooks/
│   └── ejercicio.ipynb
└── README.md

## Requisitos

- Python 3.x
- Jupyter Notebook

## Cómo ejecutar

1. Clonar el repositorio:
   git clone <link-del-repo>

2. Entrar a la carpeta:
   cd competencia_cocina

3. Abrir el notebook en Jupyter o VS Code

4. Ejecutar todas las celdas (Run All)

## Funcionalidades

- Cálculo de puntajes por ronda
- Determinación de ganadores
- Tabla de posiciones acumulada
- Visualización de resultados
